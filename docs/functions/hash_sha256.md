# hash_sha256 (function)

Generates the SHA-2/256-bit hash of a string.

Maps to the [`corefunc.HashSHA256()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#HashSHA256) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

```terraform
output "sha256" {
  value = provider::corefunc::hash_sha256("hello world")
}

#=> b94d27b9934d3e08a52e52d7da7dabfac484efe37a5380ee9088f7ace2efcde9
```

## Arguments


1. `input` (String) The string to generate an SHA-2/256-bit hash for.
