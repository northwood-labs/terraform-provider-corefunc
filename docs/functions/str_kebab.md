# str_kebab (function)

Maps to the [`corefunc.StrKebab()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/corefunc#StrKebab) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).
