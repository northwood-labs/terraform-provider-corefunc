# hash_sha3x512_base64 (function)

Generates the SHA-3/512-bit hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.Base64HashSHA3x512()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#Base64HashSHA3x512) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).
