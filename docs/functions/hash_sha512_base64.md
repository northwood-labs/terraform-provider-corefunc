# hash_sha512_base64 (function)

Generates the SHA-2/512-bit hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.HashSHA512()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#HashSHA512) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Signature

<!-- signature generated by tfplugindocs -->
```text
hash_sha512_base64(input string) string
```

## Example Usage

```terraform
output "sha512_base64" {
  value = provider::corefunc::hash_sha512_base64("hello world")
}

```

## Arguments


1. `input` (String) The string to generate a SHA-2/512-bit hash for.
