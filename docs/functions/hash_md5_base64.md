# hash_md5_base64 (function)

Generates the MD5 hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.Base64HashMD5()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#Base64HashMD5) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

```text
hash_md5_base64(input string) string
```

```terraform
output "md5_base64" {
  value = provider::corefunc::hash_md5_base64("hello world")
}

#=> XrY7u+Ae7tCTyyK7j1rNww==
```

## Arguments


1. `input` (String) The string to generate an MD5 hash for.
