# str_contains (function)

Checks if a string contains a given substring.

-> This is identical to the built-in `strcontains` function
which was added in Terraform 1.3. This provides a 1:1 implementation
that can be used with Terratest or other Go code, as well as with
OpenTofu and Terraform going all the way back to v1.0.

Maps to the [`corefunc.StrContains()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#StrContains) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Signature

<!-- signature generated by tfplugindocs -->
```text
str_contains(input string, substr string) bool
```

## Example Usage

```terraform
output "str_contains" {
  value = provider::corefunc::str_contains("hello world", "ello")
}

#=> true
```

## Arguments

1. `input` (String) The string to check.
1. `substr` (String) The substring to look for.
