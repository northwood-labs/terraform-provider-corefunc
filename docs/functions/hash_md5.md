# hash_md5 (function)

Generates the MD5 hash of a string.

-> This is identical to the built-in `md5` function, but is
available as a Go function for Terratest.

Maps to the [`corefunc.HashMD5()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#HashMD5) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Signature

<!-- signature generated by tfplugindocs -->
```text
hash_md5(input string) string
```

## Example Usage

```terraform
output "md5" {
  value = provider::corefunc::hash_md5("hello world")
}

#=> 5eb63bbbe01eeed093cb22bb8f5acdc3
```

## Arguments


1. `input` (String) The string to generate an MD5 hash for.
