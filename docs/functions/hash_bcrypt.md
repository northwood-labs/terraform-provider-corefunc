# hash_bcrypt (function)

Generates the Bcrypt hash of a string with its associated salt value.

Maps to the [`corefunc.HashBcrypt()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#HashBcrypt) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Signature

<!-- signature generated by tfplugindocs -->
```text
hash_bcrypt(input string, cost number) string
```

## Example Usage

```terraform
output "hash_bcrypt" {
  value = provider::corefunc::hash_bcrypt("hello world", 10)
}

#=> $2a$10$SOMESALTVALUEu1pYx0pG7xO6dXJv1bG8y9qzE6jFzW8u9K5eV6Zy
```

## Arguments

1. `input` (String) The string to generate the Bcrypt hash for.
1. `cost` (Number) How much effort to spend generating a value. Common values are 10-14. The default value is 10.
