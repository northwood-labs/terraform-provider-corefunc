# corefunc_hash_sha384_base64 (Data Source)

Generates the SHA-2/384-bit hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.Base64HashSHA384()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#Base64HashSHA384) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Example Usage

```terraform
data "corefunc_hash_sha384_base64" "sha384_base64" {
  input = "hello world"
}

#=> /b2OdaZ/KfcBpOBAOF4uI5hjA+oQI5IRr5B/y7g1eLPkF8txzmRu/QgZ3YwIjeG9
```
