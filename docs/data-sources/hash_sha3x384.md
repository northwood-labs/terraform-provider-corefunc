# corefunc_hash_sha3x384 (Data Source)

Generates the SHA-3/384-bit hash of a string.

Maps to the [`corefunc.HashSHA3x384()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#HashSHA3x384) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).
