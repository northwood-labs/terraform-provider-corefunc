# corefunc_hash_sha3x256_base64 (Data Source)

Generates the SHA-3/256-bit hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.Base64HashSHA3x256()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#Base64HashSHA3x256) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).
