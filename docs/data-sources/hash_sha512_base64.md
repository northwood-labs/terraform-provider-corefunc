# corefunc_hash_sha512_base64 (Data Source)

Generates the SHA-2/512-bit hash of a string. Returns a Base64 value instead of a hexadecimal string.

Maps to the [`corefunc.Base64HashSHA512()`](https://pkg.go.dev/github.com/northwood-labs/terraform-provider-corefunc/v2/corefunc#Base64HashSHA512) Go method, which can be used in [Terratest](https://terratest.gruntwork.io).

## Example Usage

```terraform
data "corefunc_hash_sha512_base64" "sha512_base64" {
  input = "hello world"
}

#=> MJ7MSJwS1utMxA9QyQLytNDtd+5RGnx6m808qG1M2G+YndNbxf9JlnDaNCVbRbDP2DDoH2Bdz33FVC6TrpzXbw==
```
